SOBRE MIM

São Paulo, Brasil

Meu nome é Victor Oliveira, tenho 22 anos e sou de São Paulo. Formado como Desenvolvedor Full Stack pela Generation Brasil. Durante minha formação aprendi desde semântica HTML até consumo de API's locais e públicas, JavaScript, ReactJS e React com TypeScript, CSS puro, e a linguagem Java para back-end, metodologia SCRUM, Git e GitHub e trabalho em equipe com git, e desenvolvi alguns projetos academicamente para adquirir mais conhecimento sobre a programação, entre eles: Blog Pessoal, E-commerce de games, Landing page, Portifolio, e outros. Junto de todas essas hard skills, também foi desenvolvida várias soft skills, como, gestão de tempo, produtividade, formação de novos hábitos, comunicação clara e assertiva, colaboração e autonomia. Minha primeira experiencia na area de TI foi como estagiario de service desk, onde adquiri conhecimento com: Avaya Agent Desktop, Avaya Communicator, Hyper-V, VMware Horizon, AnyDesk, TeamViewer, VPN Access Manager, TightVNC, M-Files, Windows Active Directory, Philips Tasy HTML / Philips Tasy Delphi, Qualitor Web (Gerenciamento de Chamados), Office 365 Administrator / Portal Office, SonicWall Firewall. Atualmento faço graduação no curso de Análise e Desenvolvimento de Sistemas, pela faculdade UNINOVE, onde estou no 2° semestre e pretendo iniciar minha carreira na programação como desenvolvedor.
